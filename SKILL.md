# weather-query 天气查询技能

查询中国城市实时天气及未来 7 天预报，包含体感温度、空气质量、生活指数等。

## 使用方法

```bash
# 查询当前天气（含体感温度）
python3 {baseDir}/scripts/weather.py 城市名

# 查询未来 7 天预报
python3 {baseDir}/scripts/weather.py 城市名 --forecast 7

# 输出原始 JSON
python3 {baseDir}/scripts/weather.py 城市名 --json
```

## 参数说明

| 参数 | 说明 |
|------|------|
| `city` | 城市名称，支持省市区，如：北京、上海市、深圳、广西灵山 |
| `--forecast` | 获取未来 N 天预报（默认 7 天） |
| `--json` | 输出原始 JSON 数据 |
| `--simple` | 仅输出简要信息 |

## 返回信息

**当前天气：**
- 温度、体感温度
- 天气状况（晴/多云/阴/雨等）
- 湿度、风向风力
- 空气质量指数 (AQI)
- 气压、能见度
- 日出日落时间

**未来预报：**
- 每日最高/最低温度
- 天气状况
- 风向风力
- 降水概率

**生活指数：**
- 穿衣建议
- 洗车指数
- 运动指数
- 紫外线指数
- 过敏指数
- 感冒指数

## 数据来源

- 中国天气网 (weather.com.cn)
- 国家气象中心 (nmc.cn)
- 广西天气网 (gxweather.com)

## 注意事项

- API 免费使用，无需密钥
- 支持中国 3000+ 城市
- 8-15 天预报具有不确定性，仅供参考
- 建议出行前查看最新短期预报
